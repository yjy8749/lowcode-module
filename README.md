# 欢迎使用 芋道-低代码

## 什么是芋道-低代码？

**芋道-低代码** 是基于 [芋道源码](https://doc.iocoder.cn/) **ruoyi-vue-pro** 框架开发的系统模块，旨在为项目快速引入低代码开发能力，显著提升业务功能的交付效率。

该模块包含以下核心功能：

- **查询器：** 通过简单的 XML 配置即可生成通用的数据查询接口，避免重复编写基础 查询 逻辑。
- **设计器：** 提供可视化拖拽界面，开发者可通过组合预置组件快速搭建页面，实现“一次开发，多次复用”。
- **集成器：** 支持在开发、测试、生产等不同环境中同步查询器与设计器的配置数据，简化部署与发布流程。

> 💡 **典型场景：** 当需要为运营人员快速上线一个列表页面时，只需在设计器中拖拽表格组件，绑定查询器定义的接口，即可生成完整页面，无需编写前后端代码。

## 什么是芋道源码？

[芋道源码](https://doc.iocoder.cn/) **ruoyi-vue-pro** 是一个现代化、易上手的 Spring Boot 开发平台。作为 [若依（RuoYi）](https://doc.ruoyi.vip/) 的全新 Pro 版本，它对原有功能进行了全面优化与重构。

技术栈采用主流组合：
- **后端：** Spring Boot + MyBatis Plus + Spring Security
- **前端：** Vue 3 / Vue 2 + Element Plus / Element UI
- **移动端：** UniApp（支持微信小程序等）

内置丰富企业级功能，包括：
- RBAC 动态权限控制
- 数据权限与 SaaS 多租户架构
- Flowable 工作流引擎
- 三方登录（微信、支付宝等）
- 支付、短信、商城等业务模块

支持多种数据库：MySQL、PostgreSQL、Oracle、SQL Server、达梦（DM）、TiDB 等。

## 什么是若依？

[若依（RuoYi）](https://doc.ruoyi.vip/) 是一个经典的企业级后台管理系统脚手架，采用稳定可靠的技术组合：
- Spring Boot
- Apache Shiro（安全框架）
- MyBatis（持久层）
- Thymeleaf（服务端渲染模板）

其核心目标是**让开发者聚焦业务逻辑**，降低技术门槛，从而节省人力成本、缩短项目周期，并保障系统安全性与可维护性。
